# M6.16 Live Frame RefTargetExtractor 归档

**任务**
- DebugTools 新增 Live Frame 入口，将实时相机帧直接喂给 RefTargetExtractor。
- 保持提取逻辑纯函数、性能不阻塞主线程。

**实现摘要**
- `RefTargetExtractor` 新增 `extract(pixelBuffer:orientation:)` 重载，直通 VisionPipeline。
- Live Debug 按钮改为从 `TierScheduler` 获取最新帧并直接调用新入口，跳过 CGImage 转换。
- Debug 输出格式统一为 `target.*`。

**性能关键**
- 移除 PixelBuffer → CGImage → PixelBuffer 双重转换。
- Live 触发仍在后台 `Task.detached` 执行，避免阻塞 UI。

- 真机：DebugTools → `M6.16 Run On Live Frame` 无卡顿，输出 `target.*`。
